# 🚀 QueryAI

An AI-powered real-time chat application built with the MERN Stack, Socket.IO, and Google's Gemini AI. Users can securely authenticate, create chat sessions, and interact with an AI assistant in real time.

## 🌐 Live Demo

Frontend: https://query-ai-frontend-black.vercel.app

## ✨ Features

- 🔐 JWT Authentication
- 👤 User Registration & Login
- 🤖 Gemini AI Integration
- 💬 Real-time Chat using Socket.IO
- 📝 Automatic Chat Title Generation
- 📂 Chat History
- 📧 Email Verification / OTP Support
- 🔒 Protected Routes
- 📱 Responsive UI
- ☁️ Deployed on Vercel & Render
